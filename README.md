# Selenium Webdriver TestNG Framework 
 Automation testing for amazon.com e-commerce website using Selenium webdriver and TestNG framework, and ExtentReports ITestListeners for generating reports.
# Prerequisites
* Eclipse IDE
* JDK (version 21+)
# Dependencies
* Selenium 
* TestNG
* AventStack
* org.slf4j
# Libraries 
* Apache Commons.io
